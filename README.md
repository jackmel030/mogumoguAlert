# mogumoguAlert

Pikmin Bloom 毒蘑菇重生計時器 PWA。打完蘑菇後截圖，自動辨識倒數時間並計算重生提醒。

## 功能

- 截圖 OCR 自動辨識剩餘時間、蘑菇類型（小 / 一般 / 巨大）
- 自動計算倒數：`剩餘時間 + 5 分鐘重生 - 30 秒預留`
- 時間到推送通知提醒
- 最多同時 10 組倒數計時
- 過期 10 分鐘自動清除，新增時自動替換最舊的已完成計時器
- PWA 支援，可加到手機桌面離線使用

## 使用教學

### Step 1：開啟蘑菇畫面並截圖

在 Pikmin Bloom 中找到毒蘑菇，進入蘑菇詳情頁面後截圖：

<img src="docs/screenshot-example.png" width="300" alt="Pikmin Bloom 蘑菇截圖範例">

畫面中包含 OCR 會擷取的資訊：
- **剩餘時間**（左下角「剩下 9 分 54 秒」）
- **蘑菇類型**（中間「一般 毒蘑菇」）

### Step 2：上傳截圖

點擊「截圖辨識」按鈕，上傳剛才的截圖（也支援直接貼上 Ctrl+V / Cmd+V）。

### Step 3：自動開始計時

OCR 辨識完成後會自動建立計時器，不需要手動確認：

- 名稱自動編號（菇1、菇2、菇3...）
- 時間自動計算（範例：9分54秒 + 5分 - 30秒 = **14分24秒**）
- 蘑菇類型自動偵測

### Step 4：等待通知

時間到時會收到推送通知「菇長回來了！」，提醒你去打蘑菇。

## 手動新增

也可以點「+ 新增計時器」手動選擇蘑菇類型和倒數時間。

## iOS 使用者注意

iPhone / iPad 必須先將網頁「加入主畫面」才能收到推送通知：

1. 用 Safari 開啟本頁面
2. 點底部 ⬆️ 分享按鈕
3. 選擇「加入主畫面」
4. 從主畫面開啟後點「開啟通知」

## 技術

- 純前端 Vanilla JS，無框架
- Tesseract.js（瀏覽器端 OCR，支援繁體中文 + 英文）
- Service Worker + Web App Manifest（PWA 離線 + 通知）
- localStorage 持久化計時器資料
